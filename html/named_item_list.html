named_item_list {LICHospitalR}R Documentation

Named list from a grouped tibble

Description

Takes in a data.frame/tibble and creates a named list that can be used in conjunction with something like save_to_excel()

Usage

named_item_list(.data, .grouping_var)

Arguments

.data

The grouped data.frame/tibble

.grouping_var

The column that contains the grouping variable

Details

  • Needs a grouped data.frame/tibble

Author(s)

Steven P. Sanderson II, MPH

Examples

## Not run: 
named_item_list(.data = df, .grouping_var = service_line)

## End(Not run)
