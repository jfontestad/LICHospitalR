oppe_cpoe_trend_tbl {LICHospitalR}R Documentation

OPPE CPOE Provider Order Trend Table

Description

Gets the oppe cpoe order trend table for all providers

Usage

oppe_cpoe_trend_tbl(.data)

Arguments

.data

The data from the workflow of oppe_cpoe_query() %>% oppe_cpoe_tbl()

Details

Takes in the data from the oppe_cpoe_query() and oppe_cpoe_tbl() functions.

Value

A tibble

Author(s)

Steven P. Sanderson II, MPH

Examples

oppe_cpoe_query() %>%
  oppe_cpoe_tbl() %>%
  oppe_cpoe_trend_tbl()
