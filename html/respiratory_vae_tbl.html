respiratory_vae_tbl {LICHospitalR}R Documentation

Respiratory VAE Tibble

Description

This takes data from the respiratory_vae_query() and performs the necessary calculations to see if a possible VAE has occurred.

Usage

respiratory_vae_tbl(.data)

Arguments

.data

The data passed in from respiratory_vae_query()

Details

  • Returns a tibble

  • Expects respiratory_vae_query() as the data argument

  • Makes the calculations according to the CDC NHSN VAE Calculator https://nhsn.cdc.gov/VAECalculator/vaecalc_v7.html

Value

A tibble

Author(s)

Steven P. Sanderson II, MPH
