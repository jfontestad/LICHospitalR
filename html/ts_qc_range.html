ts_qc_range {LICHospitalR}R Documentation

Time Series Range

Description

Get the range of a given set of numbers.

Usage

ts_qc_range(.value_col)

Arguments

.value_col

The time series value column, e.g. discharge counts

Details

  • Expects a vector of numbers be supplied

Value

A number

Author(s)

Steven P. Sanderson II, MPH

Examples

y <- seq(-5:5)
ts_qc_range(y)
