infection_prevention_patient_days_tbl {LICHospitalR}R Documentation

Patient Days for Infection Prevention Tibble

Description

This takes data in from the infection_prevention_patient_days_query() and performs the necessary manipulation/cleanup to get it ready to email out or use in conjunction with save_to_excel()

Usage

infection_prevention_patient_days_tbl(.data)

Arguments

.data

The data that results from infection_prevention_patient_days_query()

Details

  • Returns a tibble

  • Expects data from the infection_prevention_patient_days_query()

Value

An excel workbook object pointer

Author(s)

Steven P. Sanderson II, MPH

Examples

## Not run: 
library(tidyverse)

infection_prevention_patient_days_query() %>%
  infection_prevention_patient_days_tbl()

## End(Not run)
