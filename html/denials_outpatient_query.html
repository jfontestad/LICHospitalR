denials_outpatient_query {LICHospitalR}R Documentation

Denials by Outpatient

Description

#' Get denials for outpatient visits from smsdss.c_Softmed_Denials_Detail_v and smsmir.mir_pay where the left 4 digits of the pay code are '0974'. The data starts at January 1st, 2013

Usage

denials_outpatient_query()

Details

  • Uses the db_connect() and db_disconnect() functions

  • Queries:

  1. smsdss.c_Softmed_Denials_Detail_v

  2. smsmir.mir_pay

  • Includes the columns ptno_Num, bill_no, discharge_date and dollars_denied

  • This function is intended to be used with timetk::filter_by_time() if data needs to be filtered by time. The .date_var argument should be set equal to discharge_date

Value

A tibble

Author(s)

Steven P. Sanderson II, MPH

Examples

library(timetk)
denials_outpatient_query()
