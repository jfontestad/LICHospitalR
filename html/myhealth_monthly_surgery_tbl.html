myhealth_monthly_surgery_tbl {LICHospitalR}R Documentation

MyHealth Monthly Surgery Tibble

Description

This takes data in from the myhealth_monthly_surgery_query() and performs the necessary manipulation/cleanup to get it ready to email out or use in conjuntion with save_to_excel()

Usage

myhealth_monthly_surgery_tbl(.data)

Arguments

.data

The result of the myhealth_monthly_surgery_query()

Details

  • Returns a tibble

  • Expects data from the myhealth_monthly_surgery_query()

Value

A tibble

Author(s)

Steven P. Sanderson II, MPH

Examples

## Not run: 
library(tidyverse)

myhealth_monthly_surgery_query() %>%
  myhealth_monthly_surgery_tbl()

## End(Not run)
